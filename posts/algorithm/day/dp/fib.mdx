也就是

```md
F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
```

给定 `n` ，请计算 `F(n)` 。 

**示例 1：**

```md
输入：n = 2
输出：1
解释：F(2) = F(1) + F(0) = 1 + 0 = 1
```

**示例 2：**

```md
输入：n = 3
输出：2
解释：F(3) = F(2) + F(1) = 1 + 1 = 2
```

**示例 3：**

```md
输入：n = 4
输出：3
解释：F(4) = F(3) + F(2) = 2 + 1 = 3
```

**提示：**

- `0 <= n <= 30`


## 递归

时间复杂度：`O(n)`

空间复杂度：`O(n)`

```js
const fib = n => {
  if (n == 0) return 0;
  if (n == 1) return 1;
  return fib(n - 1) + fib(n - 2);
}
```

## 动态规划

根据动态规划五部曲：

* 确定DP数组以及每个下标的含义

```markdown
// dp[0] = 0;
// dp[1] = 1;
// dp[2] = dp[0] + dp[1] = 1;
// dp[3] = dp[2] + dp[1] = 2;
// dp[4] = dp[3] + dp[1] = 3;
```

* 确定递推公式

```markdown
// fib(n) = fib(n-1) + fib(n-2)
```

* 初始化 dp数组

```markdown
// n > 1 dp[n] = dp[n-1] + dp[n-2]
```

* 确定遍历顺序

从 `n = 2`开始遍历，`n == 1`或者`n == 0`可以直接返回对应的值

* 举例推导

```markdown
fib(2); // 1
```

结果如下

时间复杂度：`O(n)`

空间复杂度：`O(n)`

```js
const fib = n => {
  if (n === 1) return 1;
  if (n === 0) return 0;
  const dp = [0, 1];
  for (let i = 2; i <= n; i++) {
    dp[i] = dp[i - 1] + dp[i - 2];
  }
  return dp[n]
}
```

这里可以根据**dp**保存的值进行优化，可以使用两个数来保存前两个值。

```js
const fib = n => {
  if (n === 1) return 1;
  if (n === 0) return 0;
  let prev = 0;
  let next = 1;
  let cur = 0;
  for (let i = 2; i <= n; i++) {
    cur = prev + next;
    prev = next;
    next = cur;
  }
  return cur
}
```
