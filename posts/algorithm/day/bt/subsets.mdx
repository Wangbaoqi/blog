**示例 1：**

```md
输入：nums = [1,2,3]
输出：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
```

**示例 2：**

```md
输入：nums = [0]
输出：[[],[0]]
```

**提示：**

- `1 <= nums.length <= 10`
- `-10 <= nums[i] <= 10`
- `nums` 中的所有元素 **互不相同**

## 回溯

```js
const subsets_a = nums => {
  const ret = [], ans = [];
  const backTracking = (nums, i) => {
    ret.push([...ans]);
    // 剩余子元素为空 反过来就是 i >= nums.length
    if (i >= nums.length) {
      return;
    }
    for (let j = i; j < nums.length; j++) {
      ans.push(nums[j]);
      backTracking(nums, j + 1);
      ans.pop()
    }
  }
  backTracking(nums, 0)
  return ret
}
```
