# [复原 IP 地址](https://leetcode-cn.com/problems/restore-ip-addresses/description/)



> **有效 IP 地址** 正好由四个整数（每个整数位于 `0` 到 `255` 之间组成，且不能含有前导 `0`），整数之间用 `'.'` 分隔。

- 例如：`"0.1.2.201"` 和` "192.168.1.1"` 是 **有效** IP 地址，但是 `"0.011.255.245"`、`"192.168.1.312"` 和 `"192.168@1.1"` 是 **无效** IP 地址。

给定一个只包含数字的字符串 `s` ，用以表示一个 IP 地址，返回所有可能的**有效 IP 地址**，这些地址可以通过在 `s` 中插入 `'.'` 来形成。你 **不能** 重新排序或删除 `s` 中的任何数字。你可以按 **任何** 顺序返回答案。

 

**示例 1：**

```
输入：s = "25525511135"
输出：["255.255.11.135","255.255.111.35"]
```

**示例 2：**

```
输入：s = "0000"
输出：["0.0.0.0"]
```

**示例 3：**

```
输入：s = "101023"
输出：["1.0.10.23","1.0.102.3","10.1.0.23","10.10.2.3","101.0.2.3"]
```

 

**提示：**

- `1 <= s.length <= 20`
- `s` 仅由数字组成

## 题解 回溯



![](/Users/wangbaoqi/个人资源/图片素材搜集/博客/algorithm/backTracking/复原IP地址.svg)

复原**IP地址**也是一种分隔的问题

回溯三部曲：

- 递归函数参数以及返回值
- 递归函数终止条件
- 单层函数的解决过程



```js
// 切割的问题
// 1. 递归函数的参数
// 2. 递归结束的条件
// 3. 单层搜索的过程

const restoreIpAddresses_a = s => {
  s = s.split('');
  const ret = [];
  let pointNum = 0;

  const isValid = (str, s, e) => {
    if (s > e) {
      return false
    }
    // 不能有前导 0 
    if (str[s] == 0 && s != e) {
      return false;      
    }
    let num = 0;
    for (let i = s; i <= e; i++) {
      num = num * 10 + (str[i] - '0');
      if (num > 255) {
        return false
      }
    }
    return true;
  }

  const backTracking = (s, i, pointNum) => {
    // 递归结束的条件
    if (pointNum === 3) {
      if (isValid(s, i, s.length - 1)) {
        ret.push(s.join(''))
      }
      return;
    }
    // 单层函数搜索的过程
    for (let j = i; j < s.length; j++) {
      if (isValid(s, i, j)) {
        s.splice(j + 1, 0, '.');
        backTracking(s, j + 2, pointNum + 1);
        s.splice(j + 1, 1);
      } else {
        break;
      }
    }
  }
  backTracking(s, 0, pointNum);
  return ret
}
```
