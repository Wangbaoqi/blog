**LeetCode** [无重复字符的最长子串](https://leetcode-cn.com/problems/longest-substring-without-repeating-characters/description/)

> 给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串** 的长度。

**示例 1:**

```
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
```

**示例 2:**

```
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
```

**示例 3:**

```
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```
 
**提示：**

- `0 <= s.length <= 5 * 104`
- `s` 由英文字母、数字、符号和空格组成

## 滑动窗口

```js
const lengthOfLongestSubstring_a = s => {
  let slow = 0, fast = 0;
  let map = {};
  let max = 0;
  while (fast < s.length) {
    const fItem = s[fast++];
    map[fItem] ? map[fItem]++ : (map[fItem] = 1);

    while (map[fItem] > 1) {
      const sItem = s[slow];
      map[sItem]--;
      slow++
    }
    max = Math.max(max, fast - slow)
  }
  return max;
}
```
