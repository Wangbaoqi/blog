[有效的括号](https://leetcode-cn.com/problems/valid-parentheses/description/)



> 给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

**示例 1：**

```
输入：s = "()"
输出：true
```

**示例 2：**

```
输入：s = "()[]{}"
输出：true
```

**示例 3：**

```
输入：s = "(]"
输出：false
```

**示例 4：**

```
输入：s = "([)]"
输出：false
```

**示例 5：**

```
输入：s = "{[]}"
输出：true
```

 

**提示：**

- `1 <= s.length <= 104`
- `s` 仅由括号 `'()[]{}'` 组成

## 题解

### 借用栈实现

首先定义一个哈希表，记录括号的匹配，注意这个键值是相反的，原因是顺序遍历字符串，从括号的第二位来查找匹配值。

执行步骤：例子`([{}])`

- 首先将`(`、`[ `和`{ `入栈
- 碰到`}`，则对比栈顶元素是否跟`}`对应的哈希值相同，如果不同，则直接返回；相同，则弹出栈顶元素
- 直达字符串循环结束

时间复杂度：`O(n)`，整个字符串遍历了一遍

空间复杂度：`O(n+6)`，`6`代表哈希表中的字符集

```js
const isValid = s => {
  const map = { ')': '(', ']': '[', '}': '{' }
  const len = s.length;
  const stack = [];
  for (let i = 0; i < len; i++){
    const el = s[i];
    if (map[el]) {
      if (stack[stack.length - 1] != map[el]) return false;
      stack.pop()
    } else {
      stack.push(el)
    }
  }
  return !stack.length
}
```
