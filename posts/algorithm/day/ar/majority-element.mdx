## 解题方法

### 哈希表

首先遍历一遍数组，计算每一个数字出现的次数，存在哈希表中

然后遍历哈希表，然后得出最大出现的数子

```js
const majorityElement = nums => {
  const map = new Map();
  let max;
  let maxNum;
  for (const val of nums) {
    if (!map.has(val)) {
      map.set(val, 1);
    } else {
      map.set(val, map.get(val) + 1);
    }
  }
  for (const key of map.keys()) {
    const val = map.get(key);
    if (!max || (val >= max)) {
      max = val;
      maxNum = key;
    } 
  }
  return maxNum;
}
```

### 排序

```js
const majorityElement = nums => {
  nums.sort((a, b) => a - b)
  return nums[~~(nums.length/2)]
}
```

### **Boyer-Moore** 投票算法



```js
const majorityElement = nums => {
  let count = 0;
  let candidate;
  for (const num of nums) {
    if (count == 0) {
      candidate = num
    }
    count += (candidate == num) ? 1 : -1;
  }
  return candidate;
}
```
